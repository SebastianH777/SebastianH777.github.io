Proyectos

Bomberman

En el primer semestre en el curso de PVJ. hice un juego en python, el cual era Bomberman. Este es un juego en donde tienes que eliminar a todos los enemigos del área dejando bombas en una zona laberíntica, después de hacerlo tienes que encontrar la puerta hasta el siguiente nivel usando las bombas para destruir los bloques.

Base de Datos de una tienda de abarrotes

Este es un proyecto de mi semestre actual, este trata de una base de datos de una tienda de abarrotes la cual almacena datos de la tienda, clientes y proveedores. Con este proyecto buscamos tener una mejor manejo de las ganacias y de las ventas, para una mayor produccion.

Más sobre mi carrera

Entre estas disciplinas, destacan el Desarrollo de Algoritmos, Inteligencia Artificial, Ciencia de Datos, Ingeniería de Software, Teoría de la Computación, Programación, Robótica y Tecnologías de la Información.
